phpstorm plugin

how to install phpstorm plugin

1.preferences -> 查找live templates;->看是否有自己的live  templates;【可以验证是否有自己的 live templates】
2.  如无 cd ~/Library/Preferences/WebIde80/templates  【Templates can be found here:】
3. 将对应的xml拷贝于此；对应的XML可以在github 中去找；如有忽律；
附：https://plugins.jetbrains.com  等应用商店的bootstrap并不能
使用；只需要在其中定位到github后将livetemplate文件夹中的xml拷贝到上面的文件夹；重启IDE就可以了！
4.验证 bs3-【cmd+j就可以有提示了！】【对应的快捷键可以在ide中的live template中查看已经自定义等】
